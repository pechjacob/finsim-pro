# Adding an Event

FinSim Pro allows you to model your financial future by adding **Events** (Income, Expenses) and **Effects** (Interest).

## Steps

1.  **Select an Account:** Ensure the correct account is selected in the sidebar (e.g., "Checking").
2.  **Click "Add Income" or "Add Expense":** Located in the sidebar.
3.  **Configure the Event:**
    *   **Name:** Give it a descriptive name (e.g., "Salary").
    *   **Amount:** Enter the value.
    *   **Frequency:** Choose how often it occurs (One-time, Monthly, Recurring).
    *   **Dates:** Set the Start and End dates.
4.  **Save:** Click the checkmark icon to save.

![Adding an Event](/img/add_event_screenshot.png)

## Event Types

*   **Income:** Adds to your balance.
*   **Expense:** Subtracts from your balance.
*   **Transfer:** Moves money between accounts.
*   **Interest Effect:** Applies a percentage growth (or loss) over time.
