# Accounts

FinSim Pro supports multiple accounts, allowing you to model your entire financial portfolio in one place.

## Managing Accounts

### creating an Account
1.  Click the **+** button next to the account tabs at the bottom of the screen.
2.  Enter a name for the new account (e.g., "Savings", "Investment").
3.  The new account starts with a $0 balance.

### Switching Accounts
Click on the tabs at the bottom left to switch between accounts. Each account has its own independent:
*   Starting Balance
*   Events (Income/Expenses)
*   Simulation Chart

### Account Settings
In the sidebar, you can:
*   **Rename:** Click the account name to edit it.
*   **Initial Balance:** Set the starting amount for the account.
*   **Interest Rate:** Set a base APY (Annual Percentage Yield) for the account.

### Deleting an Account
Click the **Trash** icon in the sidebar to delete the currently selected account.
> **Warning:** This action cannot be undone.
